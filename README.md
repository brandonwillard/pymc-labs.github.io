# PyMC Labs Official Website

## Get Setup

### Environment

```bash
git clone git@github.com:pymc-labs/pymc-labs.github.io.git
cd pymc-labs.github.io
conda env create -f environment.yml  # can use mamba too!
```

### Start Lektor Server

```bash
lektor server
```

### Navigation

Edit `databags/nav.json`.
